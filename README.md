# Getting started with Go

[Go by Example](https://gobyexample.com/)


## Install Go Compiler (in Ubuntu)

```console
sudo apt install golang
```

## Run

Build and run the program in one shot:

```console
go run src/main.go
```

## Build

```console
go build src/main.go
```
